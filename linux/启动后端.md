# 启动后端服务

## 1. 创建 Dockerfile

```shell
touch Dockerfile
```

## 2. 编写 Dockerfile

```dockerfile
# 使用轻量级 OpenJDK 镜像
FROM openjdk:17

# 设置容器内工作目录
WORKDIR /app

# 从宿主机复制 JAR 包到镜像（路径需与宿主机一致）
COPY xzt-admin.jar /app/xzt-admin.jar

# 声明应用监听端口（假设为 8080）
EXPOSE 8080

# 启动命令
CMD ["java", "-jar", "/app/xzt-admin.jar", "--spring.profiles.active=prod"]
```

## 3. 构建镜像

```shell
docker build -t myapp:latest .
```

## 4. 运行 Docker 镜像

```shell
docker run -p 8080:8080 --network app-net -t -d myapp:latest
```

## 5. 查看日志

```shell
docker logs <container_id>
```
