
## 1.第一题（填空题）

小蓝一觉醒来，发现自已被困在一座高耸的塔中。这座塔共有 2025 层，每一层都刻有一个数字的立方值，从底层的 1³、2³、3³、···，一直到顶层的 2025³，层层叠叠，直入云霄。塔顶有一扇门，门旁刻着一行字：“若想离开此塔，需找出这些立方数中个位数字为 3 的数的个数。”小蓝非常着急，因为他需要尽快离开这座塔，去参加即将到来的蓝桥杯比赛。时间紧迫，请你帮助他解答这个问题。

代码实现：

```java

public class Main {
	
	public static void start() {
		int n=0;
		for(int i=1;i<=2025;i++) {
			if(i%10==7) {
				n++;
			}
		}
		System.out.println(n);
	}
	
	
	public static void main(String[]args) {
		start();
	}
}

```

## 2.第二题（编程题）

小蓝在塔的第二层发现了一面魔镜。当小蓝站在镜子前时，镜中浮现出了一行字：“将你心中的字符串输入，我将判断它是否为回文字符串，并将其反转。若你成功，便可进入下一层。”

代码实现：

```java
import java.util.Scanner;

public class Main {

    // 判断是否为回文字符串
    public static boolean isPalindrome(String str) {
        int left = 0;
        int right = str.length() - 1;
        while (left < right) {
            if (str.charAt(left) != str.charAt(right)) {
                return false;
            }
            left++;
            right--;
        }
        return true;
    }

    // 反转字符串
    public static String reverseString(String str) {
        return new StringBuilder(str).reverse().toString();
    }

    public static void main(String[] args) {
        Scanner scanner = new Scanner(System.in);
        System.out.print("请输入一个字符串: ");
        String input = scanner.nextLine();

        if (isPalindrome(input)) {
            System.out.println("'" + input + "' 是一个回文字符串。");
        } else {
            System.out.println("'" + input + "' 不是一个回文字符串。");
        }

        String reversedString = reverseString(input);
        System.out.println("反转后的字符串是: " + reversedString);
        
        scanner.close();
    }
}
```

```c
#include <stdio.h>
#include <string.h>
#include <stdbool.h>

// 判断是否为回文字符串
bool isPalindrome(const char *str) {
    int left = 0;
    int right = strlen(str) - 1;
    while (left < right) {
        if (str[left] != str[right]) {
            return false;
        }
        left++;
        right--;
    }
    return true;
}

// 反转字符串
void reverseString(char *str) {
    int left = 0;
    int right = strlen(str) - 1;
    char temp;
    while (left < right) {
        temp = str[left];
        str[left] = str[right];
        str[right] = temp;
        left++;
        right--;
    }
}

int main() {
    char input[100];
    printf("请输入一个字符串: ");
    scanf("%s", input);

    if (isPalindrome(input)) {
        printf("'%s' 是一个回文字符串。\n", input);
    } else {
        printf("'%s' 不是一个回文字符串。\n", input);
    }

    reverseString(input);
    printf("反转后的字符串是: %s\n", input);

    return 0;
}
```
